Sleek author box is a fully responsive author box plugin with 40+ social media icons and supports dark mode.

== Description ==
Sleek author box is a simple and easy-to-use plugin that automatically adds an author box to your posts. It is a fully responsive author box that can be used on any website. It is a great way to display author information on your website. It supports dark mode, 40+ social media, and all the social media icons are included in the plugin.

= Features =
1. Sleek author box show author's gravatar, name, description (bio), social media icons and a link to the author's more posts
2. Support dark mode
3. Responsive design
4. Automatically adds the author box at the bottom of the post

Icons used in this plugin are from <a href="https://fontawesome.com/license/free">Font Awesome 5.15.4</a>.

== Installation ==
1. Upload `sleek-author-box` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to use Sleek Author Box? =

After you installed the plugin, you need to go to the 'Users' menu and click on 'Profile'.
> You can change the author's name, biographical info, website URL, and email information there.

To add the author's **social media information**, you need to go to the 'Sleek Author Box' section[bottom most section on the same page].

= How to display Sleek Author Box on pages or custom post types? =

The current version of Sleek Author Box is designed to be used on posts only. In future versions, it will be extended to be used on pages and custom posts type as well.

= How to add WhatsApp link to Sleek Author Box? =

To generate yor WhatsApp link, please visit <a href="https://faq.whatsapp.com/general/chats/how-to-use-click-to-chat">faq.whatsapp.com</a>(official documentation) and follow the documentation.

= How to add Telegram link to Sleek Author Box? =

To generate yor Telegram link, please visit <a href="https://telegram.org/faq#q-how-does-t-me-work">telegram.org</a>(official documentation) and follow the documentation.

= I have two author boxes, how to get rid of one of them? =

The second author box is the one that comes by default with your theme. To get rid of it, you can use custom CSS or theme options to hide it.

= Does it support dark mode? =

Yes, it does.

= Does it support RTL languages? =

The current version does not support RTL languages.
